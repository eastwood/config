# Change Log

All notable changes to the "vsc-nib" extension will be documented in this file.

## [0.1.0] - 2024-12-13

### Added
- Initial release
- RQP Auth (SSO) command - authenticate with RQP via interactive terminal
- Clone Repository command - clone repos from GitHub organization and open in new window
- Open Sales Zoom command - quickly open configured Zoom meeting
- Open Jira Ticket command - open Jira tickets with current branch as default
- Open Buildkite command - open Buildkite projects with workspace name as default
- Create Review Notes command - generate formatted PR review notes
- Configurable settings for all URLs and paths
